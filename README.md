# MoodTracker — Инструкция по установке APK

## Шаг 1 — Создай аккаунт на GitHub
Перейди на https://github.com и нажми Sign up (бесплатно).

## Шаг 2 — Создай репозиторий
1. Нажми + (правый верхний угол) → New repository
2. Название: MoodTracker
3. Нажми Create repository

## Шаг 3 — Загрузи файлы
1. Нажми Add file → Upload files
2. Перетащи ВСЕ файлы из архива (сохраняй структуру папок!)
3. Нажми Commit changes

## Шаг 4 — Жди сборку (~3-5 минут)
1. Перейди на вкладку Actions
2. Увидишь запущенный процесс "Build APK"
3. Жди зелёной галочки

## Шаг 5 — Скачай APK
1. Нажми на завершённый процесс
2. Внизу найди раздел Artifacts
3. Скачай MoodTracker-APK.zip → распакуй → внутри app-debug.apk

## Шаг 6 — Установи на телефон
1. Отправь app-debug.apk себе в Telegram
2. Открой файл на телефоне
3. Нажми Установить (разреши установку из неизвестных источников если попросит)
